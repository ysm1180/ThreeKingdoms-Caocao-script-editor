# newjojogame-script-editor

## New Jojo Game Editor
NewJojoGame : https://github.com/ysm1180/newjojogame

## Start
```
npm install gulpjs/gulp-cli -g
npm install 
npm run rebuild
```

## For development (Build & Watch)
```
npm run dev
```

## Build source
```
npm run build
```

## Package build for win64
```
npm run build:win
```
